# Task filters and compact GUI menus

## Summary

В GUI добавлены компактные верхние меню и фильтр задач. Фильтр позволяет:

- показывать завершенные, архивные и отмененные задачи;
- ограничивать выборку задач диапазоном дат;
- применять фильтр без изменения текущей вкладки Google Sheets.

При включении завершенных задач GUI автоматически включает диапазон дат за последний месяц, чтобы не загружать слишком большой набор данных по умолчанию.

## GUI flow

Верхняя панель разделена на три меню:

- `Задачи`: загрузка задач, создание рейса, обновление;
- `Обработка`: обработка всех ТС, Wialon, Я.Карты;
- `Справочники`: история, пользователи, настройки.

Кнопка фильтра открывает небольшую панель с настройками:

- `Завершённые`;
- `Даты`;
- дата начала;
- дата окончания;
- `OK` / `Отмена`.

Горячая клавиша `F5` обновляет текущую таблицу.

## API flow

Клиентский репозиторий задач хранит активные фильтры:

- `include_completed`;
- `date_from`;
- `date_to`.

Эти значения передаются в `GET /api/v1/tasks` вместе с текущим `source_key`.

На стороне API параметры пробрасываются в `PostgresTaskReader.read_page()`, где:

- незавершенные задачи остаются режимом по умолчанию;
- завершенные/архивные/отмененные задачи включаются только при `include_completed=true`;
- диапазон дат применяется к первой доступной дате задачи из `planned_start_at`, `planned_end_at`, `completed_at`.

## Refresh behavior

Когда фильтры активны, инкрементальное обновление отключается и выполняется полная постраничная загрузка. Это нужно, чтобы результат не смешивал старые строки с новым набором фильтра.

Завершенные, архивные и отмененные задачи подсвечиваются отдельным спокойным зеленым фоном. Просроченные выгрузки для таких строк не подсвечиваются.
